# Исследование рынка стартапов и инвестиционных трендов

Проект посвящен углубленному анализу исторических данных о финансировании стартапов. На основе очищенных данных и анализа динамики рынка формируются инвестиционные рекомендации на 2015 год.

## Цели и задачи проекта
* **Предобработка данных:** Очистка датасета от дубликатов, обработка пропущенных значений (включая расчет скрытых интервалов финансирования) и приведение типов данных к корректным форматам.
* **Инжиниринг признаков:** Классификация стартапов по срокам привлечения инвестиций и сегментация рынка на массовые, средние и нишевые категории.
* **Фильтрация и анализ выбросов:** Поиск и устранение аномалий с помощью метода межквартильного размаха ($IQR$) отдельно по каждому рыночному сегменту.
* **Анализ динамики:** Изучение трендов финансирования по годам (объемы, популярность раундов, нормированный возврат средств) с фокусом на изменения в 2014 году.
* **Бизнес-рекомендации:** Формирование стратегии для инвестора на 2015 год (выбор отрасли и оптимального типа финансирования).

## Стек технологий
* **Language:** Python 3
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
* **Environment:** Jupyter Notebook

## Структура исследования

Проект логически разделен на 5 ключевых этапов:

1. **Знакомство с данными и предобработка**
   * Загрузка данных из внешних источников (включая zip-архивы).
   * Исправление типов данных, заполнение пропусков в `mid_funding_at` расчетным путем.
   * Удаление неинформативных строк и дубликатов.
2. **Инжиниринг признаков (Feature Engineering)**
   * Группировка компаний по длительности инвестиционных циклов.
   * Категоризация сегментов рынка (`mass`, `mid`, `niche`) на основе объема представленных компаний.
3. **Работа с выбросами и фильтрация**
   * Расчет аномального финансирования через $IQR$.
   * Отсечение неполных периодов и фильтрация данных по активности рынка (отбор лет с $\ge 50$ инвестиционными раундами).
4. **Анализ динамики рынка**
   * Оценка популярности и денежного объема различных типов финансирования (`seed`, `venture`, `angel` и др.).
   * Анализ динамики изменения объемов раундов вплоть до конца 2014 года.
   * Расчет нормированного показателя возврата средств (ROI-метрика).
5. **Итоговые выводы и рекомендации**
   * Аналитический прогноз и рекомендации для заказчика на 2015 год.

## Ключевые результаты и инсайты
* **Инвестиционные рекомендации на 2015 год:** На основе анализа динамики 2013–2014 гг. сформирован топ-3 перспективных секторов для входа: *Payments* (стабильность), *Hardware + Software* (взрывной рост) и *Medical Devices* (надежность).
* **Оптимальный инструмент:** Венчурное финансирование (`venture`) определено как приоритетное благодаря максимальному объему привлеченного капитала и лучшим показателям возврата средств.
* **Качественная очистка данных:** С помощью метода $IQR$ (межквартильный размах) из анализа были исключены аномальные сделки-выбросы, что позволило оценить реальные, типичные аппетиты рынка.
* **Масштабируемость выводов:** Доказано, что фокус инвестора должен быть направлен на массовые сегменты, так как долгосрочные сделки в них генерируют основной объем возвращаемых средств.
